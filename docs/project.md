Project Name: SemanticSQL

AI-powered Natural Language Interface for RDBMS.

Users interact with a relational database using plain English instead of SQL.

Examples:
- Show all employees with salary above 50000
- Create employee table with 5 columns

Features:
- Natural Language to SQL
- SQL Validation
- SQL Auto Correction
- Semantic Cache using Redis
- Database Explorer
- Query Analytics
- Query History
- System Monitoring
- Model Management

Tech Stack:

Frontend:
React
TypeScript
Material UI

Backend:
FastAPI

Database:
MySQL

Cache:
Redis

Embeddings:
Sentence Transformers
